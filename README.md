# Retail Analytics & AI-Powered Sales Forecasting System

This project analyzes retail sales data, identifies trends, builds forecasting models, performs store segmentation, and presents insights through a Streamlit dashboard.

## Tools Used
- Python
- Pandas, Matplotlib
- Scikit-learn
- Statsmodels
- Streamlit

## How to Run
1. Run notebooks in order
2. Then run:
streamlit run dashboard/app.py
